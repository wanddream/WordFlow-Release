# WordFlow 语音输入工具

中文语音识别工具，支持将语音实时转换为文字并输入到任意应用程序。

## 功能特点

- 🎤 **一键语音输入** - 按下快捷键即可开始录音
- 🇨🇳 **中文语音识别** - 基于 Paraformer 模型，识别准确率高
- ⌨️ **自动文字输入** - 识别结果自动输入到当前光标位置
- 📝 **支持任意应用** - 记事本、Word、WPS、浏览器等均可使用
- 🔒 **本地处理** - 语音识别在本地完成，无需联网，保护隐私

## 下载安装

### 系统要求

- Windows 10/11 (64位)
- 麦克风设备

### 安装步骤

1. 下载 [WordFlow-Setup.exe](https://github.com/wanddream/WordFlow-Release/releases/download/v1.0.0/WordFlow-Setup.exe)
2. 运行安装程序，选择安装位置
3. 首次运行时，程序会自动弹出模型下载向导
4. 选择模型并等待下载完成（约 200 MB）
5. 安装完成，可以开始使用！

> **提示**：如果模型下载失败，可以在向导中勾选"使用国内镜像"加速下载。

## 使用方法

### 基本操作

1. **启动程序** - 双击桌面图标或从开始菜单启动
2. **开始录音** - 按下快捷键（默认：`Caps Lock` 或 `右 Alt`）开始录音
3. **停止识别** - 再次按下快捷键停止录音并开始识别
4. **查看结果** - 识别的文字会自动输入到当前光标位置

### 系统托盘

程序运行后会在系统托盘显示图标：
- 🟢 绿色图标 - 服务已连接，可以开始录音
- 🔴 红色图标 - 服务未连接，请检查环境

右键托盘图标可以：
- 显示/隐藏主窗口
- 打开设置
- 退出程序

## 配置说明

### 快捷键设置

默认快捷键：
- `Caps Lock`（大写键）
- `右 Alt`（右侧 Alt 键）

可在设置中自定义快捷键。

| 模型 | 大小 | 说明 |
|------|------|------|
| Paraformer 中文 | ~200 MB | 中文语音识别，准确率高 |

## 常见问题

### Q: 语音识别没有反应？

1. 检查麦克风是否正常工作
2. 确认托盘图标状态（绿色表示正常）
3. 尝试重启程序

### Q: 识别结果无法输入到目标窗口？

1. 确保目标窗口处于活动状态
2. 确保光标在可输入区域
3. 尝试以管理员权限运行程序

### Q: 模型下载失败？

1. 检查网络连接
2. 在模型下载向导中勾选"使用国内镜像"
3. 手动下载模型文件到 `PythonASR/models/` 目录

### Q: 安装后无法启动？

1. 确保安装路径没有中文或特殊字符
2. 检查杀毒软件是否拦截
3. 尝试重新安装

## 技术架构

- **前端**: WPF (.NET 8)
- **语音识别**: FunASR (Paraformer) + ONNX Runtime
- **后端服务**: Python Flask
